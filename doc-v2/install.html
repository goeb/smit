Installation

Windows Platform

  • Unzip
  • smit.exe is ready to use

Linux

Requirements:

  • OpenSSL

Installation instructions:

git clone http://github.com/goeb/smit
cd smit
make

And copy the compiled executable smit to your PATH (for example /usr/bin).
