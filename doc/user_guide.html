User Guide

Getting Started

With your web browser, go to the address of the Smit server. By default the TCP listening port is 8090.

Example:

http://localhost:8090/

You need to have cookies enabled.

Signin In

In the Sign-In page, enter the username and password assigned to you by the administrator.
