Getting Started

Online demo

An online demo is available at this address: http://smit.herokuapp.com

Getting Started on Linux

git clone https://github.com/goeb/smit.git
cd smit
./configure
make
./smit serve demo &

Then, browse to: http://127.0.0.1:8090/things_to_do/issues

Getting Started on Windows

Unzip the smit package, and run:

init.bat
start.bat

By default these scripts:

  • initiate a repository with a superadmin with username admin and password admin.
  • start smit server on port 8090
  • start a browser at address http://127.0.0.1:8090
